**1.安裝 Eclipse + NDK**
============

**Step by step**
-------------

 - **安裝 Eclipse**
    - **cd 資料夾名稱 //切到要存放檔案的資料夾**
    - **wget http://dl.google.com/android/adt/22.6.2/adt-bundle-linux-x86-20140321.zip //下載android adt bundle**
    - **unzip adt-bundle-linux-x86_64-20140321.zip // 解壓縮.zip檔**
    - **nano ~/.bashrc // 設置路徑**
    - **export PATH={PATH}:~/adt-bundle-linux-x86-20140321/sdk/tools // 設置路徑**
    - **export PATH=${PATH}:~/adt-bundle-linux-x86-20140321/sdk/platform-tools // 設置路徑**
    - **參考資料 : [How to install Android SDK on Debian 7 Wheezy**](http://siddou.hd.free.fr/2014/04/how-to-install-android-sdk-on-debian-7-wheezy/)
     
 - **安裝 Android NDK**
    - **cd 資料夾名稱 //切到要存放檔案的資料夾**
    - **wget http://dl.google.com/android/ndk/android-ndk-r9d-linux-x86.tar.bz2 //下載Android NDK**
    - **tar jxvf android-ndk-r9d-linux-x86.tar.bz2 // 解壓縮tar.bz2檔**
